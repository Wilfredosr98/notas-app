Tecnologías más usadas

Node JS

Node. js es un entorno JavaScript que nos permite ejecutar en el servidor, de manera asíncrona, con una arquitectura orientada a eventos y basado en el motor V8 de Google. Es una plataforma que avanza muy rápidamente y cada vez está más presente en el mercado.

Python

Este lenguaje simplifica mucho la programación. Es un gran lenguaje para scripting, si usted requiere algo rápido (en el sentido de la ejecución del lenguaje), con unas cuantas líneas ya está resuelto. Elegante y flexible. Es una gran multiplataforma con frameworks de gran utilidad.

Wordpress

Representa el 39,5% de todos los sitios web. Aunque es conocido principalmente por sus blogs, este sistema de gestión de contenidos (CMS) gratuito puede servir para una gran variedad de sitios web, como tiendas de comercio electrónico, portafolios y portales de información.

BOT

Creación de un bot para un grupo de trabajo en discord de la pagina APPEN, este bot realiza Web Scrapping a la pagina y consulta si existe una tarea nueva, el numero de tareas disponibles, el nivel y el pago en centavos de la tarea, si alguna de estas variables cambia el bot lo notifica al discord. Fue desarrollado a partir de Python y se encuentra en el servidor de Heroku funcionando.

APLICACION DE NOTAS

Aplicación web Backend y Frontend, utilizando Nodejs, Mongodb, Express y otras tecnologías de Javascript. Es una aplicacion de Notas, CRUD de nodejs, más un Login y Registro utilizando Mongodb, protege nuestras rutas del servidor, se encuentra estilizada con CSS y Bootstrap4.
